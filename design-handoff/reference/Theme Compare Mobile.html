Locked direction

  • Party header = 3-cell strip. Three columns of name + role + HP bar. Active player gets accent left-border + “turn” caption. Wins for legibility when HP changes round-to-round.
  • Dice = inline chip. Lives in the message flow between bubbles, like a scene beat. Format: die · check · result · PASS/FAIL.
  • Crimson Void = toned-down palette. Tokens: --gold #c8232e → #b2222d, --gold-bright #ff5a5f → #e85257, --border-gold #6a1c24 → #5a1820. FX: GM glow alpha 70%→40%, ember-pulse peak ~60%, backdrop opacity 0.85→0.72, cycle 5.5s→6.5s.
