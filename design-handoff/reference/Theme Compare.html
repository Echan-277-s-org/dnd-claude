What changed v1 → v2 → Crimson

  • Theme B re-skinned as “Crimson Void.” Original dark-order sci-fi villain palette — not a branded recreation. Black + crimson + ember on black; tokens map 1:1 to the existing names (--gold = crimson primary, --border-gold = blood-line, --gold-bright = ember).
  • FX swap. Starfield → sparse ember dust + a slow crimson pool. Holo scanlines → a much fainter (0.06) interlace plus a hot inner ember on the GM bubble. holoFlick opacity stutter → emberPulse 5.5s box-shadow breathing.
  • Corners went from HUD brackets to faceted triangles (◆ ◄ ► shapes per corner).
  • Setup card itself is faceted — corners chamfered via clip-path, mirroring the Sith-plate aesthetic without copying any insignia.
  • Copy rewritten as an original dark-order operation (“Obsidian Harrow”) so it reads villainous without leaning on trademarked names.

Worth a look

  • The ember-pulse on the emblem may read as a distraction during long sessions — consider gating it behind “reduced motion” or limiting to setup-only.
  • Crimson on near-black is high-contrast; the GM bubble glow can crowd at small widths. If it overpowers, drop the outer box-shadow alpha from 70% → ~45%.
  • The faceted setup-card clip removes its border-radius entirely; if you want both, layer a faceted background plate behind a rounded inner card.
